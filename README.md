# Upload-Required-Document

Student want to join OIL-Laboratory in Ming chi University of Technology Taiwan need to upload the personal document in this repository. The document must in English language includes:

1. Curriculum vitae

2. Study plan 

3. English certificate (TOEFL, IELTS, TOIC, and etc.) 

4. Project report include the code

   Students who have experience in making projects using programming languages can upload reports and code here. This aims to determine and show up the ability of students in making projects using software. 

Please **put the document in different folder**,  to make it easier for review. 

If you don't know how to upload your document, please follow [the guideline here](https://github.com/oilmcut2020/required-document/blob/master/asset/Upload-document.md)
